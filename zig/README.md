# Zig

## Example command

To run the compiled program on Linux:

```bash
$ ./meta -s ../sequences/meta.txt -d ../sequences/db.txt -k 17 -a 1
```

## Compile

To compile the program on Linux:

```bash
$ zig build-exe -O ReleaseFast main.zig --name meta
```
